# 1) *Start/mode* panel

![panel](design-1.jpg)

This panel is composed of:
- a key switch (`P1_SW-KEY`) for the on/off
- a green led (`P1_LED`) for the on/off
- a 3-position switch (`P1_SW-3POS`) for the mode (spaceship/game/computer)
